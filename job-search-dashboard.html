GitHub Pages setup guide
1

Create a GitHub account

Go to github.com and sign up for a free account if you don't have one.

2

Create a new repository

Click the + icon → New repository. Name it job-search. Set it to Public. Check "Add a README file". Click Create repository.

3

Upload this file

In your new repo, click "Add file" → "Upload files". Upload the job-search-dashboard.html file you downloaded. Then rename it to index.html by clicking the file → pencil icon → change the filename at the top.

4

Enable GitHub Pages

Go to your repo → Settings → Pages (left sidebar). Under "Source", select "Deploy from a branch". Choose main branch, / (root) folder. Click Save. Your dashboard will be live at:

https://YOUR-USERNAME.github.io/job-search
5

Add your API key

Open your live dashboard URL. Go to Setup tab. Paste your Anthropic API key from console.anthropic.com and click Save. The key is stored in your browser only.

6

Bookmark and use

Bookmark the URL. Every Monday and Friday, open it and click "Find new roles". Your pipeline data persists in localStorage — it will be there every time you return.
